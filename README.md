# 🚀 Job Application Tracker Dashboard

A full-stack MERN (MongoDB, Express.js, React.js, Node.js) web application that helps users track and manage their job applications efficiently through a modern analytics dashboard.

This project includes authentication, multi-user support, application analytics, search/filter functionality, charts, and a responsive dark-themed UI.

---

# ✨ Features

## 🔐 Authentication System
- User Registration
- User Login
- JWT Authentication
- Secure Password Hashing using bcrypt

## 📋 Job Application Management
- Add Applications
- Update Application Status
- Delete Applications
- Track Notes and Roles

## 📊 Dashboard Analytics
- Total Applications Count
- Interview Count
- Rejected Count
- Selected Count

## 📈 Data Visualization
- Interactive Pie Chart using Chart.js
- Application Status Distribution

## 🔎 Search & Filter
- Search by Company Name
- Filter by Application Status

## 👤 Multi-User Architecture
- Separate dashboards for each user
- User-specific application data storage

## 🎨 Modern UI
- Responsive Dark Theme
- Modern Dashboard Design
- Smooth Hover Effects
- Professional SaaS-style Layout
